Senior Exhibition Website

The brief

Create a single page website for the 2017 senior design exhibition using the guidelines established by a separate branding committee.

Research

Wireframes

Each member of the committee came up with their own wireframes. We then critiqued and then combined them to create our final wireframe. This wireframe was flushed out in illustrator and then written in HTML.

Final Mockup

The final mockup was arrived at after multiple iterations. Creating the final static mockup was done in conjuction with in browser experiments. The slider that moves the letters up and down was done quickly and early one evening to sort out what was possible in our narrow time frame. In browser experimentation was especially important to sort out where we wanted to integrate motion into the page. At one point, I had a swarm of letters flocking around the web page which was a little bit much, but that experiment resulted in the idea to pattern the background with letters instead.

Implementation

Technologies Used

  • Gulp.js
  • Simple carousel
  • Susy
  • SASS
  • SMACSS (CSS organization)

I built the website using gulp as the task runner, you can see my starter template here. I use gulp to automatically reload the browser on file changes (browser sync), convert SASS, convert nunjucks templates, minify CSS/JS/Images, and auto prefix CSS.

For a single page website with a fairly simple grid system, I decided to use Susy, which is a small SCSS grid making toolkit. The SCSS was organized according to the SMACSS principles.
